# 趋势决策台

面向手机端的趋势交易研究 H5。公开地址：

https://lucia-trend.github.io/trend-decision-radar/

## 简单使用说明

1. 点击右上角“登录 / 存档”，注册用户名和密码。未登录时观察池只保存在当前设备；登录后会同步到 Cloudflare D1。
2. 在“观察池”搜索 AI ETF、科创 AI、创业板 AI、MSTR 等品种，点击 `+` 加入，最多保存 20 个。
3. 回到“总览”点击“同步”。页面会同时读取市场状态和观察池截面，并生成三条个性化趋势纪律摘要。
4. “信号”“ETF”“观察池巡检”和“拆解”只在点击时调用趋势动物 API；全量穿透会先估费再执行。
5. 市场温度、趋势强度、右侧状态和官方信号属于接口事实；恐贪、高开、高走指数与纪律摘要属于页面策略推导。

## 数据与安全

- 趋势动物 API Key 仅保存在 Cloudflare Worker Secret，不进入前端或仓库。
- Worker 限制 GitHub Pages 来源、请求类型和请求大小。
- 密码采用带随机盐的 PBKDF2-SHA-256 摘要保存；服务端不保存明文密码。
- 会话使用随机令牌，D1 只保存令牌摘要；登录失败和注册操作设有限流。
- 用户可在账号面板删除账号及云端观察池。

